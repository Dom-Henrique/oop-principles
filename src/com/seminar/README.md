# Seminar Management System
## Requirements
- Register students, professors, seminars and local to realize
- Student only be in the one seminar
- Professor can ministrate many seminars
- Seminar need have a local
## Variables
- Seminar: title, date;
- Student: name, age;
- Professor: name, area;
- Local: address.
